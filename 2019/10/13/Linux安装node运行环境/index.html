Linux搭建node运行环境

发表于 2019-10-13 分类于 Linux ， node

最近在一直找一个API管理服务，无意间看到了DOCLever,试用了一下，功能很强大，完全满足我们的需求，而且还是开源可以部署在内网中。刚开始，在自己的mac电脑中搭建，本身早就装好了node环境，后来到Linux上后，不是所有的环境都有Node运行环境,所以就开始了Node安装之路。

ln -s /home/weblogic/node-v6.11.4-linux-x64/bin/node /usr/local/bin/node
